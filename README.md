# The Backrooms Minecraft Mod

A Fabric mod that brings the unsettling liminal spaces of The Backrooms to Minecraft.

## Setup

This mod requires Java 17 and Minecraft 1.20.1.

### Building

Run the following command to build the mod:

```bash
./gradlew build
```

On Windows:

```bash
gradlew.bat build
```

The built mod file will be in `build/libs/`.

## Development

- Mod ID: `backrooms`
- Package: `com.backrooms`
- Main Class: `com.backrooms.BackroomsMod`
- Client Class: `com.backrooms.client.BackroomsClient`
